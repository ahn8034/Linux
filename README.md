###ps

###kill
- 현재 실행중인 프로세스를 강제 종료하므로 명령어로 옵션이 몇개 있음

###top

###htop

* http://hisham.hm/htop/

###tree

###vi
- 에디터이며, vim이라고 vi의 개선판이 있으며 리눅스 개발자들은 별도로 설치해서 사용하는 경우가 있음
* i, ESC - 편집모드, 편집탈출
* h,j,k,l - 방향키(편집모드 탈출시에만 동작)
* CTRL + a - 제일 마지막 글자로 이동하여 편집 모드 전환
* dd - 한줄 삭제
* a - 현재 위치에서 삽입 모드
* SHIFT + y,p - 한줄 복사
* CTRL + f - 다음 페이지 이동
* CTRL + b - 이전 페이지 이동
* http://dany-it.tistory.com/18 참고할 것
